# Automation Overview

The Automation Overview dashboard provides a high-level summary of your automated test runs, helping you gauge the overall health, stability, and efficiency of your testing efforts.

## Key Metrics
- **Total Test Runs**: The total number of test executions over the selected period.
- **Pass Rate**: The percentage of tests that passed across all runs.
- **Flakiness Factor**: The percentage of tests that exhibit unstable behavior.
- **Average Duration**: The average time it takes for a test run to complete.

## Trend Analysis
The dashboard includes interactive charts that illustrate testing trends over time. 
- Track Pass Rates day by day or week by week.
- Identify drops in stability before they affect your release candidate.
- Monitor total test volume and how execution time fluctuates.

## Readiness Score
The Readiness Score is an AI-driven metric that calculates your current release confidence. It takes into account pass rates, flakiness, recent failures, and coverage metrics to give you a clear "Go/No-Go" indication for production deployments.

## Quick Actions
From the Overview, you can quickly jump into **Recent Runs** to see the latest execution details or go to **Automation Insights** to analyze infrastructure or flaky test issues.
